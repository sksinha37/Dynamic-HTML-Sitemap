Dynamic HTML Sitemap provides a flexible and customizable HTML sitemap for WordPress websites, including pages, posts, and categories.

== Description ==
Dynamic HTML Sitemap generates a customizable HTML sitemap for your WordPress website, including pages, posts, custom post types, and categories. The plugin allows you to exclude items from the sitemap via the admin panel, and offers a fully customizable design.

== Installation ==
1. Upload the `dynamic-html-sitemap` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[dynamic_sitemap]` shortcode to display your sitemap.

== Frequently Asked Questions ==
= How do I exclude items from the sitemap? =
In the plugin settings, you can select pages, posts, or custom post types to exclude from the sitemap.
